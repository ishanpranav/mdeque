Class MDeque<T>

A linear collection that supports element insertion and removal at three points: front, middle and back. The name m-deque is short for "double ended queue" (deque) with m for "middle" and is pronounced "em-deck". M-deque has no fixed limits on the number of elements it contains.

Remarks

The remove operations all return null values if the m-deque is empty. The structure does not allow null as an element.

All pop..., push..., and peek... operations (from all three points of access) are constant time operations.

The middle position is defined as (size+1)/2. For m-deques with odd sizes, this is the actual middle element. For the m-deques with even sizes, this is the element, one past the middle of that elements. The position count is zero-based.
